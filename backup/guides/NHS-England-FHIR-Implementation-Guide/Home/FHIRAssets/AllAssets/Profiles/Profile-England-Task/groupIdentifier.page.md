## `groupIdentifier`

<b>Definition:</b><br>

EPS Only.

An identifier to the original prescription-order MUST be present. Only the short form identifier needs to be supplied.

```json
"groupIdentifier": {
        "system": "https://fhir.nhs.uk/Id/prescription-order-number",
        "value": "88AF6C-C81007-00001C"
    },
```
