## `providedBy`

<b>Definition:</b>

It is recommended an *identifier reference* using ODS code is used. 

```json
"providedBy": {
        "identifier": {
            "system": "https://fhir.nhs.uk/Id/ods-organization-code",
            "value": "RBA"
          },
        "display": "TAUNTON AND SOMERSET NHS FOUNDATION TRUST"
    }
```
