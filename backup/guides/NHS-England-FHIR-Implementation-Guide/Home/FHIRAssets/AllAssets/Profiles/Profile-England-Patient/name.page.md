## `name`

<b>Definition:</b>

At least one patient name must be provided. It is recommended a use is provided.

```json
"name":  [
  {
    "family": "Smith",
    "given":  [
      "Jane"
    ],
    "prefix":  [
      "Mrs"
    ],
    "suffix":  [
      "MBE"
    ],
    "use": "usual"
  }
]
```
