## `subject`

<b>Definition</b><br>


A resource reference to a Patient with a traced NHS Number is required, an untraced NHS Number **MUST NOT** be used.

```json
 "subject": {
                    "type": "Patient",
                    "reference": "urn:uuid:bde9eba6-079f-4210-8108-6ea8db58de8c",
                    "display": "Miss Bernie Kanfeld"
                }
```
