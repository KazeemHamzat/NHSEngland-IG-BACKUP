## `extension:portalLink`

<b>Definition:</b><br>

Deep Link into the specific landing page in MYR.

```json
"extension": [
        {
            "url": "https://fhir.nhs.uk/StructureDefinition/Extension-Portal-Link",
            "valueUrl": "https://virtual.nhs.uk/MyPatientRecords"
        }
    ],
```
